== Installation ==

= Installation =
1. In your WordPress Dashboard go to "Plugins" -> "Add Plugin".
2. Search for "Web2application".
3. Install the plugin by pressing the "Install" button.
4. Activate the plugin by pressing the "Activate" button.
5. Go to "Web2application" Settings section on the wordpress admin panel and enter your API key (there is instruction in the page).
6. Go to "Web2application" Send Push section and send you notification to all your users

= Minimum Requirements =
* WordPress version 4.0 or greater.
* PHP version 5.2.4 or greater.
* MySQL version 5.0 or greater.

= Recommended Requirements =
